# 🌦️ Modern Weather Dashboard

A modern and responsive **Weather Dashboard** built using **HTML, CSS, and JavaScript** that provides real-time weather information for any city around the world. The application fetches live weather data using the **WeatherAPI** and displays detailed weather information in a clean, user-friendly interface.

Whether you're checking the weather for your current location or searching for another city, the dashboard delivers accurate weather updates along with a **3-day forecast**, making it a simple and reliable weather application.

---

## ✨ Features

- 🌍 Search weather by city name
- 📍 Automatic current location detection
- 🌡️ Real-time temperature
- 🤗 Feels Like temperature
- 💧 Humidity
- 🌬️ Wind Speed
- ☀️ UV Index
- 🌫️ Air Quality Index (AQI)
- 🕒 Local Time
- ☁️ Dynamic weather condition icons
- 📅 3-Day Weather Forecast
- 🌙 Dark Mode
- ⌨️ Search using the Enter key
- ⏳ Loading animation while fetching data
- ❌ Error handling for invalid city names
- 📱 Fully responsive design

---

## 📖 How It Works

1. Enter a city name or allow location access.
2. The application sends a request to the WeatherAPI.
3. Weather data is retrieved and processed.
4. Current weather details and a **3-day forecast** are displayed on the dashboard.
5. Users can switch between light and dark mode for a better viewing experience.
